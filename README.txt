
To run path finding algorithm:
STEP	1:	click left mouse key on the box to put starting point
STEP	2: 	Again click left mouse key on the box to put ending point
STEP	3:	Again click left mouse key to place block
   Additional Step: Click right mouse key to remove clicked boxes to place them again 
STEP 	4: 	Press a key in keyboard
To stop: 
	click 'X' on the top right corner of window
To re-run the algorithm:
	Press s key in keyboard

There might be some bugs but will be removed soon
